# ☁️ AZ-104: Azure Resource Manager (ARM) Templates Lab

Bem-vindo ao meu repositório de estudos práticos de nuvem! 🚀

Este projeto é um laboratório de **Infraestrutura como Código (IaC)** focado na implantação e gerenciamento de recursos no Microsoft Azure utilizando **ARM Templates (JSON)**. A interface foi documentada e estilizada utilizando HTML5 e CSS3 para facilitar a leitura, revisão dos comandos e fixação do conteúdo.

## 🎯 Objetivo do Projeto

Aprofundar os conhecimentos em administração de infraestrutura cloud rumo à **certificação Microsoft AZ-104**. A transição para a automação e implantação via templates é o próximo passo após a consolidação dos fundamentos (AZ-900). Esse domínio permite escalar recursos com precisão, habilidade que considero essencial no dia a dia de operações de TI.

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Microsoft Azure:** Implantação de Storage Accounts e gerenciamento de Resource Groups.
* **Azure CLI & Azure PowerShell:** Autenticação e execução de scripts de implantação.
* **JSON:** Estruturação lógica dos templates ARM, uso de parâmetros (`parameters`), variáveis e outputs.
* **HTML5 & CSS3:** Criação da interface web interativa para documentar o passo a passo do laboratório.

## ⚙️ O que você vai encontrar aqui?

O arquivo principal (`index.html`) atua como um portal de documentação detalhando:
1. Comandos de autenticação e criação de grupos de recursos.
2. A lógica de implantação usando a variável `$templateFile`.
3. Testes práticos com passagem de parâmetros (ex: alteração dinâmica de SKUs de armazenamento para `Standard_GRS` e `Basic`).
4. O código-fonte final do modelo `azuredeploy.json` totalmente configurado.
